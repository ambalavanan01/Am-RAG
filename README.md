# Professional Document AI (Local RAG)

A powerful, robust, and visually elegant Retrieval-Augmented Generation (RAG) application built with Python, Streamlit, LangChain, FAISS, and OpenRouter. 

This application allows users to upload documents (PDF, DOCX, PPTX, TXT) to their local machine, automatically chunk and embed the text into a completely local vector database using HuggingFace models, and "chat" with their documents seamlessly.

## Features
- **Strictly Local Storage:** Your uploaded documents and vector embeddings never leave your computer.
- **Multi-Format Support:** Easily ingests PDFs, Word documents, PowerPoints, and plain text files.
- **Lightning Fast Vector Search:** Uses Facebook's FAISS (Facebook AI Similarity Search) and `sentence-transformers` for instant, offline semantic retrieval.
- **OpenRouter LLM Integration:** Uses the state-of-the-art Llama 3.3 70B Instruct model via OpenRouter to generate concise, highly accurate answers.
- **Transparent Citations:** Every answer generated by the AI includes expanding citation buttons so you can verify the exact document and paragraph the AI used to build its answer.
- **Premium UI:** Features a custom Light Theme overriding standard Streamlit styling, using Times New Roman typography, refined chat bubbles, drop shadows, and polished step-by-step progress indicators.

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have Python 3.10+ installed on your machine.

### 2. Installation
Clone the repository and install the required dependencies:

```bash
[git clone https://github.com/ambalavanan01/Am-RAG.git](https://github.com/ambalavanan01/Am-RAG.git)
cd Am-RAG

# Install the dependencies
pip install -r requirements.txt
```

### 3. Setup OpenRouter API Key
The application utilizes OpenRouter for the intelligence layer. 
Currently, the API key is hardcoded in `app.py`. For production or public sharing, it is highly recommended to move this mapping to a secure `.env` file instead.

### 4. Run the Application
Start the Streamlit server locally:

```bash
streamlit run app.py
```

The application will automatically launch in your default web browser at `http://localhost:8501`.

## 🛠️ Architecture
- **Frontend:** Streamlit 
- **Document Loading & Splitting:** LangChain (`PyPDFLoader`, `Docx2txtLoader`, `UnstructuredPowerPointLoader`)
- **Embeddings:** HuggingFace (`all-MiniLM-L6-v2`)
- **Vector Store:** FAISS (Locally persisting to `./faiss_index`)
- **LLM / Generation:** OpenRouter (`meta-llama/llama-3.3-70b-instruct` via `ChatOpenAI`)

## ⚠️ Security Warning
Do **not** commit your raw API keys to GitHub. Ensure keys are rotated if exposed. The uploads folder and vector database are ignored in `.gitignore` to prevent leaking personal documents.
